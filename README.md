# Вики проекта: Recommendation System

  Технологии:  
- Java 23  
- Jakarta EE  
- Spring Data JPA  
- Spring MVC  
- Lombok  
- PostgreSQL  
- Telegram Bot API  
- Swagger (OpenAPI)

  Краткое описание:  
Система рекомендаций, предоставляющая пользователям автоматизированные рекомендации, управляемые динамическими и статическими правилами. Для взаимодействия предусмотрены веб-API, Telegram-бот и интеграция с внешними системами для сброса кеша и сбора статистики.
